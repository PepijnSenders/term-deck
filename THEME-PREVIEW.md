## Quick Launch

```bash
# Launch a specific theme
./run-theme.sh matrix
./run-theme.sh neon
./run-theme.sh retro
./run-theme.sh minimal
./run-theme.sh hacker

# Or use the full command
bun bin/term-deck.ts slides-neon/
```


## Mix & Match

You can also customize individual slides! Add theme override in frontmatter:

```yaml
---
title: SPECIAL SLIDE
bigText: WOW
gradient: pink
theme: neon    # This slide uses neon theme even in matrix deck
---
```

This lets you create dramatic theme transitions mid-presentation!
